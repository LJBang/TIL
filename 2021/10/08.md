## 이번엔 DP
### 15486. 퇴사2
N+1일 되는날 무조건 퇴사한다.  
하루에 하나씩 N개의 일이 주어지고, 각 일은 처리하는 시간과 보상이 다르다.  
최대한 많은 보상을 얻기위해서 일을 할 때, 보상은?  

```py
N = int(input())
_list = []
answer = [0]*(N+1)
for _ in range(N):
    _list.append(list(map(int,input().split())))

for i in range(0,N):
    if _list[i][0] <= N-i:
        answer[i+_list[i][0]] = max(answer[i+_list[i][0]], answer[i]+_list[i][1])
    answer[i+1] = max(answer[i+1], answer[i])
        
print(answer[N])
```
